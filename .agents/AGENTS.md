# Sesotho Orthography & Spelling Guidelines

When generating new vocabulary, suggesting edits, coining translations, or writing comments/documentation, always respect modern standard/Gauteng Sesotho spelling conventions:

1. **Use 'di' instead of 'li'**:
   - Modern Sesotho represents the /di/ sound using the letter **d** instead of **l** before **i** (and **u** in active environments).
   - E.g., write *modimo* instead of *molimo*, *dikobo* instead of *likobo*, *dipalo* instead of *lipalo*.

2. **Use 'wa'/'w' instead of 'oa'/'o'**:
   - Modern Sesotho writes the semi-vocalic glide using **w** instead of **o**.
   - E.g., write *ngwana* instead of *ngoana*, *senwamadi* instead of *senoamali*, *mantswe* instead of *mantsoe*, *lentswe* instead of *lentsoe*.
