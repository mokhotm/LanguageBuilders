# Sesotho Root Dictionary for STEM Word Building
## A Reference for the LBOS Coining Engine

This dictionary serves as the morphological foundation for the LBOS word coining engine. 
It catalogs the Sesotho verbs, nouns, adjectives, and noun class prefixes available for 
constructing new STEM vocabulary.

## 4. Noun Class Prefix System

The Sesotho noun class system is the primary mechanism for creating new STEM words through nominalization. Class 7 (se-) is the most important for STEM as it denotes instruments, tools, and manners.

| Class | Prefix | Typical Use | STEM Relevance |
|:---|:---|:---|:---|
| 1 | mo- | Person / agent (singular) | Scientists, practitioners |
| 2 | ba- | People / agents (plural) | Plural of Class 1 |
| 3 | mo- | Natural phenomenon / plant | Natural science terms |
| 4 | me- | Natural phenomena (plural) | Plural of Class 3 |
| 5 | le- | Augmentative / paired | Large objects, paired things |
| 6 | ma- | Mass / collection | Collective/mass nouns |
| **7** | **se-** | **Instrument / tool / manner** | **🏆 Most useful for STEM!** |
| 8 | di- | Instruments (plural) | Plural of Class 7 |
| 9 | N-/e- | Animals / abstract / loanwords | Abstract concepts |
| 10 | di-/liN- | Animals / abstract (plural) | Plural of Class 9 |
| 14 | bo- | Abstract quality / state | Abstract sciences |
| 15 | ho- | Infinitive / verbal noun | Processes, actions |

### Common Derivation Patterns

**Verb → Instrument (Class 7)**:
- ho bala (to count) → **se-bal-i** = counting instrument (calculator/computer)
- ho tsamaisa (to drive) → **se-tsamais-i** = driving instrument (engine)
- ho bontsha (to show) → **se-bontsh-i** = showing instrument (display/screen)

**Verb → Abstract Process (Class 9/14)**:
- ho fetola (to change) → **phetoho** = change/transformation
- ho bopa (to create) → **popo** = creation
- ho kopanya (to unite) → **kopano** = union/combination

**Verb → Agent (Class 1)**:
- ho ruta (to teach) → **mo-rut-i** = teacher
- ho ithuta (to learn) → **mo-ithut-i** = learner/student
- ho fumana (to discover) → **mo-fuman-i** = discoverer/scientist

## 5. Curated STEM Word Examples

### Successfully Coined Words

| English | Sesotho | Method | Etymology |
|:---|:---|:---|:---|
| Gravity | kgohelo | Semantic Calque | ho hohela (attract) → Class 9 nasalization |
| Gravity | matlakgohedi | Compounding | matla (force) + kgohedi (attractor) |
| Engine | setsamaisi | Semantic Calque | se- (instrument) + tsamaisa (cause motion) |
| Computer | kelellomotlakase | Semantic Calque | kelello (brain) + motlakase (electricity) |
| Computer | sebali | Nominalization | se- (instrument) + bala (count) |
| Photosynthesis | popoleleseli | Semantic Calque | popo (creation) + leseli (light) |
| Molecule | karolwana | Semantic Calque | karolo (part) + -wana (diminutive) |
| **Virus** | **kokwanahloko** | **Semantic Calque** | **kokwana (tiny creature) + hloko (pain). Already used by native speakers!** |
| Virus | setshwayi | Nominalization | se- (agent) + tshwaya (infect) + -i |
| **Biology** | **thutobophelo** | **Compounding** | **thuto (study/education) + bophelo (life). Literally: "study of life"** |
| **Energy** | **matla-mosebetsi** | **Compounding** | **matla (power) + mosebetsi (work). Literally: "power to do work"** |
| **Force** | **tshusumetso** | **Semantic Calque** | **ho susumetsa (to push/impel) + Class 9 nominalization** |
| **Power** | **sekgahla sa matla** | **Compounding** | **sekgahla (rate) + sa (of) + matla (energy/power). Literally: "rate of energy"** |
