README: Bukantswe Sesotho-English Bilingual Dictionary
--------------------------------------------------------------------
Full name:	Bukantswe Sesotho-English Bilingual Dictionary
Description:	Bilingual English-Sesotho dictionary
Language:	Sesotho & English
Version:	3.0
Size:		276 Kb
--------------------------------------------------------------------

This dataset represents a basic Sesotho dictionary compiled in the creation of a Sesotho language resource. The dictionary was developed by Prof. J.A.K. Olivier and is based on an on line word list published and revised since 1996.

The resource consists of 10085 bilingual segments in Sesotho and English.
